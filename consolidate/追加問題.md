Q  git addの取り消し

>全てのファイルを取り消し
**git rm --cached -r**

>特定のファイルのみを取り消し
**git rm --cached -r ファイル名**

- 2回目以降git addを取り消す
>全てのファイルを取り消し
**git reset HEAD**

>特定のファイルのみを取り消し
**git reset HEAD ファイル名**

**Gitの動作原理が関係していて、2回目以降のコマンドが違うのは分かったけど、具体的なことが分からない**
